# 🌐 Home Assistant Add-on: Seerr Server 🚀

Welcome to the Seerr Server for Home Assistant repository! This add-on is designed to host Seerr independently within your Home Assistant environment, providing easy and direct access to your self-hosted media request system.

🎉 Features
Independent Hosting: Seamlessly runs Seerr as a standalone service within Home Assistant.
Direct Access: Seerr is available at http://<your-home-assistant-ip>:5055.
Persistent Storage: Configurations and data are preserved across restarts.
Continuous Data Sync: Regular synchronization ensures data integrity and up-to-date information.
💡 About Seerr

Seerr is a versatile, self-hosted application that allows you to manage movie and TV show requests in one place. It integrates with platforms like Plex and Jellyfin, automating the process of handling and approving content requests.

Customize it to create a simple and user-friendly interface for friends or family to request content without needing direct access to your media server.

🚀 Quick Start
Install the Add-on: Add this add-on to your Home Assistant instance from the add-on store.
Run: Start the add-on to get Seerr up and running on port 5055.
Access: Open your browser and navigate to http://<your-home-assistant-ip>:5055 to access your Seerr dashboard.

Your files are safely backed up to /share/seerr in your Home Assistant installation.

🛠 Installation
Open your Home Assistant Supervisor panel and go to the Add-on store.

Add this repository URL to your add-on store or you can click this button

Install the Seerr Server add-on.
Start the add-on and check the logs to ensure everything is running smoothly.
Access Seerr at http://<your-home-assistant-ip>:5055.
Updating

Whenever there's an update to Seerr, you can try updating the add-on by rebuilding it on the add-on page in Home Assistant. However, there is no guarantee that it will work. Please open an issue if you face difficulties after upgrading.
